## 🚀 Features

- **🌐 Multilingual**: Supports both **Spanish** and **English** versions.
- **🎨 Custom Design**: A clean, modern look using **Glassmorphism** and a curated dark theme palette (I really love red!).
- **⚡ Performance**: Built as a static site using the Next.js App Router for optimal speed.
- **📱 Responsive**: Fully adapted for all screen sizes.

## 🛠 Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Icons**: [React Icons](https://react-icons.github.io/react-icons/)
- **Deployment**: [Cloudflare Pages](https://pages.cloudflare.com/)
